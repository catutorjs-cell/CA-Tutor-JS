# CA Foundation – PDF Files Guide

Place all CA **Foundation** level PDF papers in the appropriate subfolder below.

---

## 📁 Folder Structure

```
pdfs/
└── foundation/
    ├── pyq/                    ← Previous Year Question Papers (PYQs)
    ├── rtpmtp/
    │   ├── rtp/                ← Revision Test Papers (RTPs)
    │   └── mtp/                ← Mock Test Papers (MTPs)
    └── answers/
```
